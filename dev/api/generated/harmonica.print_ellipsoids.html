harmonica.print_ellipsoids¶

harmonica.print_ellipsoids(**kwargs)[source]¶

Print all available ellipsoids.

These are the ones that are hard-coded into the library. You can always set your own ellipsoid using harmonica.set_ellipsoid and harmonica.ReferenceEllipsoid.

Any keyword arguments given to this function will be passed to print.

Examples

>>> print_ellipsoids() # doctest: +ELLIPSIS
ReferenceEllipsoid(name='GRS80', ... long_name='Geodetic Reference System 1980')
ReferenceEllipsoid(name='WGS84', ... long_name='World Geodetic System 1984')
